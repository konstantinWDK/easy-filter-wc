A simple and powerful WooCommerce product filter plugin with category, tag, price, and attribute filtering.

== Description ==

Easy Filter WC is a lightweight and user-friendly WooCommerce product filter plugin that allows your customers to easily find products by filtering through categories, tags, price ranges, and product attributes.

**Key Features:**

* Filter by product categories
* Filter by product tags  
* Price range filtering with slider or input fields
* Filter by product attributes
* AJAX-powered filtering for smooth user experience
* Responsive design that works on all devices
* Easy shortcode implementation: `[easy_filter]`
* Customizable admin settings panel
* Works with any WooCommerce-compatible theme

**How to Use:**

1. Install and activate the plugin
2. Go to Settings > Easy Filter WC to configure your filter options
3. Use the shortcode `[easy_filter]` in your sidebar, widget, or anywhere on your site
4. Customers can now filter products and see results instantly

**Perfect for:**

* Online stores with large product catalogs
* WooCommerce sites needing better product discovery
* Shops with multiple product categories and attributes
* Sites wanting to improve user experience

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/easy-filter-wc/` directory, or install through WordPress admin
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings > Easy Filter WC to configure the plugin
4. Use the `[easy_filter]` shortcode to display the filter

== Frequently Asked Questions ==

= Does this plugin work with all themes? =

Yes, Easy Filter WC is designed to work with any WooCommerce-compatible WordPress theme.

= Can I customize the appearance of the filter? =

Yes, you can customize the styling by modifying the CSS or using custom CSS in your theme.

= Does the plugin support AJAX filtering? =

Yes, the plugin includes AJAX filtering for a smooth, page-refresh-free experience.

= Can I filter by custom product attributes? =

Yes, the plugin automatically detects and includes all your WooCommerce product attributes.
